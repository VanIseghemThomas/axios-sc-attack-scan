# Axios supply chain attack scanner
A simple scanning utility script to check if your system or any or your projects are compromised as a result of the Axios supply chain attack.

Based on the following articles:
- [Aikido](https://www.aikido.dev/blog/axios-npm-compromised-maintainer-hijacked-rat)
- [SafeDep](https://safedep.io/axios-npm-supply-chain-compromise/)

**Important:** will only run on Linux / MacOS.

## Usage

```
./scan.sh <path-to-scan>
```

This makes it easy to scan dedicated folders that contain repositories without having to go through the whole disk.
